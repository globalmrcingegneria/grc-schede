# Schede tecniche MRC Ingegneria

Pacchetto generato il 2026-04-21 dal Gestore Schede MRC v2.

## Contenuto
- `index.html` — indice navigabile delle schede
- `schede/{id}/index.html` — pagina di destinazione di ciascuna scheda (1 totali)
- `registro.json` — snapshot dei dati sorgente (per backup e re-import)

## Pubblicazione su GitHub Pages
1. Estrai l'archivio
2. Copia i contenuti nella tua repo (es. `globalmrcingegneria/grc-schede`), sovrascrivendo le versioni precedenti
3. Commit & push
4. Dopo 1-2 minuti le pagine saranno accessibili agli URL dei QR code

## Backup
Il file `registro.json` contiene tutti i dati sorgente. Conservalo per poter reimportare il registro nel tool.
